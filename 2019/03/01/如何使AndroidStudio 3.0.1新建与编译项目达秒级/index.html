如何使AndroidStudio 3.0.1新建与编译项目达秒级

2019-03-01 AndroidStudio

AndroidStudio 3.0.1 新建与编译项目慢的主要问题
1、gradle构建项目
2、下载support包

1、gradle构建项目达秒级的解决方案

1.1、查看项目构建的gradle版本并下载

打开项目gradle文件夹下的gradle-wrapper.properties配置文件，如下图所示

其distributionUrl就是编译的过程中需要下载的gradle版本地址，这就是导致gradle编译项目慢的主要原因，我们可以将gradle-4.1-all.zip下载链接地址使用下载软件下载到本地如下：

1、 .gradle文件夹是gradle缓存环境变量的路径 ；
2、 gradle-4.1文件夹是gradle环境变量的路径；
3、 gradle-4.1-all.zip文件是gradle包的路径；

我们将gradle编译过程中的gradle下载地址改为我们本地的gradle包的地址，能使gradle编译项目达到秒级（注意Mac版和Windows版配置有区别）

1.5、配置依赖包下载地址

使用国内阿里云依赖包下载地址替换为Google依赖包下载地址，可以提高非Offline模式下快速下载项目依赖的包，使得编译项目达到秒级。

2、support包新建项目达秒级的解决方案

第一次根据自己选择的SDK版本新建项目需要下载的support包，生成一个项目并编译成功之后，将AndroidStudio配置为Offline模式，再次新建项目就能达到秒级。

将红框线中的 Offline work 复选框选中则可以使用Offline模式

至此使得AndroidStudio新建与编译项目达到秒级的gradle构建项目与support包下载的配置已完成！开始试试吧！
