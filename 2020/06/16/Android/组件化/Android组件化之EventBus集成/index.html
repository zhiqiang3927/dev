2020-06-16 Android组件化

组件化项目数据的通信机制，使用操作方便的EventBus

EventBus 介绍

EventBus Github 地址

EventBus优点

  • 简化组件之间的通信
  • decouples event senders and receivers
  • 对activity、fragment和background threads执行良好
  • 避免复杂和容易出错的依赖关系和生命周期问题
  • 使您的代码更简单
  • 快
  • 很小 (~60k jar)
  • 在实际安装的应用中得到验证
  • 具有高级功能，如发送线程，用户优先级等
